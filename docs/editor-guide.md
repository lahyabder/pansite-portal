# Guide Rédacteurs et Gestionnaires GED (Content/GED Editor Guide)

Bienvenue aux équipes de communication et de documentation du **Port Autonome de Nouadhibou**. Ce guide explique vos workflows principaux.

## 1. Interface et Bilinguisme

Votre site est nativement bilingue : **Français** et **Arabe (🇲🇷)**.
Presque toutes nos interfaces d'édition vous demandent d'entrer à la fois la version française et la traduction arabe. Le site détecte la langue du visiteur public et charge la base correcte. 

> _Conseil_ : Si un champ arabe est vide, le site utilisera parfois le français par défaut, mais pour des raisons SEO, traduisez l'intégralité du contenu avant la publication stricte.

---

## 2. Rédacteur de Contenus (Module "Contenus")

Accès : **Menu > 📰 Contenus**
Permissions : Créer, Mettre à jour des articles.

Ce module permet de créer :
- **Actualités** : Nouvelles régulières du PAN.
- **Communiqués** : Publications de presse à télécharger.
- **Événements** : Réceptions, salons avec dates et lieux.
- **Alertes** : Avertissements avec un indicateur (ex: Météo, Route barricadée).

### Le Cycle de Vie du Contenu
1. **Brouillon (Draft)** : À la création d'un article, cliquez sur "Enregistrer comme brouillon". L'article existe mais est privé sur le back-office.
2. **En Révision (Pending Approval)** : (Sauf si vous avez un accès direct) Le brouillon est soumis aux directeurs de comm (Rôle _Validator_).
3. **Publié (Published)** : L'article s'affiche en temps réel sur la vitrine publique `www.pan.mr`.
4. **Archivé (Archived)** : L'article n'est plus visible du public.

---

## 3. Responsable GED (Module "Documents")

Accès : **Menu > 📁 Documents GED**
Ce portail métier dématérialise toute la bibliothèque institutionnelle (Tarifs portuaires, textes de loi environnementaux, registres maritimes, etc.).

### Créer un nouveau document
Remplissez scrupuleusement les _Métadonnées_:
- **Thème** : (Réglementation, RH, Tarification, Sécurité, etc.)
- **Direction / Entité** : Pour classer par origine (Capitainerie, Dir Générale).
- **Type de fichier** / Langue du PDF.

### Contrôle d'Accès du Fichier
Il est très important de choisir le bon niveau !
- 🌐 **Public** : Tous les visiteurs du site pan.mr le trouveront par Google ou la barre "Recherche de Doc". Libre téléchargement.
- 🔒 **Restreint** / 🏢 **Interne** : Réservé aux utilisateurs ayant un compte (comme les partenaires du Port ou les employés authentifiés via ERP). Les visiteurs voient que le document "existe", mais ont besoin de s'authentifier lors du clic.

### Gestion des Versions
Un document juridique évolue. Ne supprimez **jamais** l'ancien PDF.
- Utilisez l'écran "**Gérer les versions**" (`v1.4`, `v2.0`).
- Cliquez sur "+ Nouvelle version", ajoutez un commentaire : "Mise à jour suite au décret de 2026". 
- Le système écrase l'affichage public vers ce nouveau fichier, mais garde une trace (via l'historique) de l'ancienne version.
